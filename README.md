
# 📚 Library Management System (Python)

This project is a simple Library Management System developed in Python. It is designed as a menu-driven application that allows users to manage library operations such as issuing and returning books efficiently.

## 🚀 Features
- View available books
- Issue books with student name and due date
- Return books with fine calculation
- Automatic fine system based on delay:
  - 1st week: ₹10/day/book
  - 2nd week: ₹20/day/book
  - 3rd week: ₹60/day/book (increasing pattern)
- Dictionary-based data management
- Clear and user-friendly interface

## 🛠️ Technologies Used
- Python
